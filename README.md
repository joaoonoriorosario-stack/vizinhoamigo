# Vizinho Amigo

Projeto desenvolvido para o **Concurso Agrinho 2026**, na categoria **Programação**, Subcategoria 3: **Programação Front-End com HTML, CSS e JavaScript**.

## Tema

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.**

## Slogan

**Solidariedade que chega à sua porta.**

## Sobre o projeto

O **Vizinho Amigo** é um protótipo educativo de site responsivo inspirado em uma rede comunitária de apoio.

A proposta do projeto é ajudar moradores do campo ou de comunidades mais afastadas a criarem listas de compras e necessidades. Essas listas poderiam ser visualizadas por vizinhos conhecidos e voluntários, que poderiam colaborar quando fossem se deslocar até a cidade.

Muitas famílias que vivem em áreas rurais ou em comunidades distantes precisam ir até a cidade para comprar alimentos, remédios, materiais de construção, produtos de higiene e outros itens importantes. Esse deslocamento pode ser difícil, caro ou demorado, principalmente para idosos, pessoas com mobilidade reduzida ou famílias sem transporte próprio.

O projeto mostra como a tecnologia pode ajudar a organizar a solidariedade entre pessoas próximas, fortalecendo o cuidado comunitário e a relação entre campo, cidade e comunidade.

## Objetivo

O objetivo do projeto é demonstrar, de forma educativa, como um site pode ser usado para organizar listas de compras e necessidades, aproximar vizinhos conhecidos e fortalecer redes de apoio comunitário.

O protótipo não realiza cadastro real, pagamento, entrega ou transação financeira. Ele apenas simula a ideia para fins educacionais.

## Funcionalidades

O site possui:

* escolha entre perfil de morador e perfil de vizinho voluntário;
* página inicial com apresentação da proposta;
* criação de lista de compras e necessidades;
* remoção de itens da lista;
* botão de envio habilitado somente quando há itens cadastrados;
* simulação de contribuição simbólica em reais;
* confirmação de lista enviada;
* listas simuladas da comunidade;
* oferta fictícia de ajuda por um vizinho voluntário;
* chat simulado sem uso de servidor;
* perfil educativo com dados fictícios;
* rank de confiança simulado;
* histórico de ajuda e cadastro comunitário simulados;
* seção sobre o projeto e sua relação com o Agrinho 2026;
* salvamento da lista no navegador;
* layout responsivo para celular, tablet e computador.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

O projeto foi desenvolvido com tecnologias básicas de front-end, sem uso de frameworks, bibliotecas externas ou CDN.

O recurso `localStorage` foi utilizado para salvar informações no próprio navegador do usuário. Esse recurso é nativo do navegador e não depende de biblioteca externa.

## Como executar o projeto

Para abrir o projeto localmente:

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Abra o arquivo `index.html` em um navegador.

O site também pode ser publicado no GitHub Pages, pois não precisa de servidor, banco de dados ou instalação de pacotes.

## Como funciona a simulação

O usuário pode escolher um perfil para navegar pelo protótipo.

No perfil de morador, é possível criar uma lista de compras e necessidades, adicionando itens como alimentos, remédios, produtos de higiene ou materiais de uso cotidiano.

No perfil de vizinho voluntário, o site mostra listas simuladas da comunidade. O usuário pode escolher uma lista, oferecer ajuda e enviar mensagens em um chat fictício.

As informações apresentadas são apenas exemplos educativos e não representam pessoas reais.

## Confiança comunitária

O projeto apresenta uma simulação de rank de confiança, histórico de ajuda e cadastro comunitário.

Essa ideia foi incluída para mostrar que uma rede de apoio precisa de responsabilidade, cuidado e vínculo entre as pessoas. Em uma aplicação real, seria necessário ter validação dos usuários, proteção de dados, prevenção de fraudes e acompanhamento da comunidade.

No protótipo, esses recursos são apenas demonstrativos.

## Acessibilidade

O projeto foi desenvolvido buscando facilitar a navegação e a leitura. Para isso, foram utilizados:

* estrutura semântica em HTML;
* textos alternativos em imagens informativas;
* botões grandes e com nomes claros;
* contraste entre texto e fundo;
* foco visível nos elementos interativos;
* layout adaptado para diferentes tamanhos de tela;
* navegação simples no computador e no celular;
* campos com identificação adequada.

## Relação com o Agrinho 2026

O projeto se relaciona com o tema do Agrinho 2026 ao tratar da conexão entre campo, cidade, tecnologia e comunidade.

Uma comunidade sustentável não depende apenas da produção e do cuidado ambiental. Ela também precisa de cooperação, organização, acesso a itens essenciais e apoio entre as pessoas.

O **Vizinho Amigo** mostra como a tecnologia pode ajudar a fortalecer vínculos comunitários e facilitar ações simples de solidariedade no dia a dia.

## Sustentabilidade social

O projeto trabalha a ideia de sustentabilidade social, entendida como o cuidado com as pessoas e com a vida em comunidade.

Ao organizar pedidos e aproximar vizinhos, a proposta pode inspirar atitudes de colaboração, reduzir o isolamento de moradores e facilitar o acesso a produtos importantes, principalmente em regiões afastadas dos centros urbanos.

## Conceitos de programação utilizados

Durante o desenvolvimento do projeto, foram aplicados conceitos como:

* estruturação de páginas com HTML;
* estilização com CSS externo;
* responsividade;
* manipulação do DOM;

* funções em JavaScript;
* arrays e objetos para dados simulados;

* renderização dinâmica de listas;

## Principais interações implementadas

O site permite:

* escolher um perfil de navegação;
* voltar para a tela inicial;
* alternar abas;
* adicionar e remover itens;
* exibir a lista de compras e necessidades;
* mostrar ou ocultar o campo de contribuição simbólica;
* enviar e limpar uma lista;
* ajudar com uma lista como vizinho voluntário;
* enviar mensagens em um chat simulado;
* atualizar visualmente a navegação ativa;
* navegar pelo menu no desktop e pela navegação inferior no celular.
